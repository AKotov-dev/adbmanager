IconExtractor.apk - это дополнительный пакет для ADBManager. После установки пакета на ваш смартфон, ADBManager сможет использовать оригинальные иконки приложений. Список приложений выводится медленнее, но управление приложениями становится более удобным. На устройствах Android_6-10 вам потребуется предоставить разрешение на доступ к хранилищу мультимедиа.

IconExtractor.apk is an add-on package for ADBManager. After installing the package on your smartphone, ADBManager will be able to use the original app icons. The list of applications is displayed more slowly, but managing applications becomes more convenient. On Android_6-10 devices, you will need to grant permission to access media storage.
